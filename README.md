# Liberation

超抜報告を登録する Rails アプリケーションです。

## Setup

`mise` を使う場合は、最初にこのプロジェクトを trust してください。

```bash
mise trust
mise install
mise run setup
```

## Development

Rails サーバーの起動:

```bash
mise run server
```

テスト実行:

```bash
mise run test
```

## Environment

- Ruby `3.4.8`
- Bundler gems are installed into `vendor/bundle`
